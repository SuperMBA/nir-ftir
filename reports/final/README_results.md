# Итоговые результаты, включённые в репозиторий

Эта папка содержит компактные итоговые таблицы и финальные рисунки, которые сопровождают магистерскую ВКР по аугментации ATR-FTIR спектров биологических жидкостей.

Полные run-level отчёты, промежуточные артефакты и тяжёлые папки экспериментов не версионируются, чтобы не перегружать репозиторий. В Git включены только итоговые таблицы и figure-ready графики, достаточные для проверки основных выводов.

## 1. GDB small-n: PCA / factor–PC associations

- `gdb_dimdesc_window_summary.csv` — сводка по спектральным окнам и изменению factor–PC associations после аугментации.
- `gdb_dimdesc_best_pc_per_factor.csv` — best-PC таблица по клиническим факторам.

Основной вывод: на малом наборе GDB small-n классическая аугментация наиболее заметно меняет геометрию данных в локальном окне Amide III, тогда как на более широких спектральных диапазонах эффект существенно слабее.

## 2. GDB small-n: synthetic-data QC

- `gdb_qc_amide3_method_summary.csv`
- `gdb_qc_amide3_label_method_summary.csv`
- `gdb_qc_broad_method_summary.csv`
- `gdb_qc_broad_label_method_summary.csv`

Эти таблицы используются для контроля качества синтетических данных и сравнения baseline / classic augmentation / VAE в малом клиническом наборе.

Основной вывод: синтетические данные нельзя оценивать только визуально или только по росту одной supervised-метрики. Необходим контроль качества синтетики: real-vs-synth AUC, kNN overlap, Wasserstein distance и downstream sanity-checks.

## 3. Diabetes saliva

- `diabetes_meta_only_holdout.csv` — компактная сводка по diabetes saliva.

Этот набор используется как более крупный saliva-контраст к GDB small-n. Он помогает показать масштаб-зависимый характер эффекта аугментации: на более крупных наборах аугментация может работать как регуляризатор supervised-моделей, тогда как на very small-n основной интерес смещается к изменению геометрии данных и риску ложного оптимизма.

## 4. Финальные рисунки

Основные figure-ready графики лежат в `reports/figs/`:

- `fig1_dimdesc_windows.png` / `.pdf` — сравнение спектральных окон по PCA / factor–PC associations.
- `fig2_pc_curve_amide3_Anamnes_factor.png` / `.pdf` — пример распределения связи клинического фактора с PCA-компонентами в Amide III.

## 5. Методический смысл

Репозиторий отражает не единичный запуск, а воспроизводимый исследовательский пайплайн:

- baseline → augmentation сравнение;
- train-only augmentation;
- фиксированные preprocessing-профили;
- повторные запуски по seed;
- supervised-метрики и калибровка;
- geometry-first анализ для small-n;
- synthetic-data QC.

Итоговый вывод: аугментация ATR-FTIR спектров должна рассматриваться как методологическая интервенция в структуру данных, а не как автоматический способ улучшить классификационные метрики.
